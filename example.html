Implementation with example using Brid Player This example (live sample here) is using the Brid.TV player, but the concept is the same for other players. Create "video slots" using the relevantDigital.defineVideoSlots() function. They should be create once on the page. The same GAM ad-unit-path can be used multiple times, for example if you want to have two pre-roll videos. Call relevantDigital.loadPrebid() as normal. Use the relevantDigital.loadVideoUrls(slotIds, callback) function to wait for loading of an array of slots. The callback will get a corresponding array of VAST-URLs that you can use with the player.
